LIB·OF·BABEL

a walkable Library of Babel

An endless library you can walk. Each hexagonal gallery has four walls of bookshelves (700 books) and two hallways to the next gallery, plus a staircase up and down — four moves from anywhere.

Nothing is stored. Every gallery and book is generated on the spot from its coordinates, so the same address always holds the same books, forever, for everyone. The Library is infinite yet weighs almost nothing: only your path is remembered in the browser.

Name a universe in the header to step into a parallel infinite library — same rooms, wholly different books. Blank is the default universe. Pick an alphabet (Borges 25 or Basile 29) and you enter yet another distinct library: new text, new hashes, new titles.

This guide has four tabs — ← → switch them while it is open.

Inspiration

Jorge Luis Borges, “The Library of Babel” (1941).
Jonathan Basile, libraryofbabel.info

How to wander

  • LIB·OF·BABEL (header) opens this guide anytime.
  • Use the minimap — click an exit hash to move. arrow keys walk hallways and stairs (unless a dialog is open).
  • Click gallery (z, n) to jump anywhere on the lattice.
  • Each gallery has a sigil — an emblem from its hash. Same place, same sigil; click to download the SVG.
  • Type a universe name or roll ⚄ for a random one; blank returns to default.
  • The alphabet menu switches libraries entirely — new spines, text, and fingerprints.
  • actions… → copy link (or click the header hash) for a permalink to this spot.
  • actions… → search… — pick content or title, find coordinates, jump there (see more tab).
  • window · last 50 (footer) revisits recent galleries; export journey saves the full path as JSON.

Reading a book

  • Click any colored spine to open it; ← → turn pages while the reader is open.
  • The spine you last closed is outlined in red so you can find it again on this shelf.
  • color redraws the page as a character-colour map — each symbol a square, tinted by the gallery hash.
  • link copies a permalink to this book at the current page.
  • save… → borrow book — full text export (~1.3 MB .txt)
    → book image — whole book as a PNG colour map

The numbers in the header

gallery (z, n) is your coordinate. hash is a BLAKE3 fingerprint of this gallery's 700 books — a permalink and proof, identical every time you return (and unique to the current universe and alphabet). rarity counts leading zero bits in that hash. Uncommon galleries (8+ bits) are saved to trophies automatically as you walk. Your last 50 galleries are kept in the history window; every step's hash is kept in the full trail. actions… also offers verify journey, prospect for rarity, and new walk.

Search by content

In actions… → search…, leave the dropdown on content. Paste any phrase (up to one full book — ~1.3 million characters) and the library returns the gallery, shelf, book, and page(s) where it already lives — then opens it. Long phrases span consecutive pages in the same volume. Search uses the alphabet selected in the header and stays in the universe you are standing in. Invalid characters are highlighted in red.

Search by title

Same search dialog — choose title instead of content. Type a spine title (up to 24 characters, same alphabet rules). The library finds the gallery and shelf where that title belongs, embeds it on the canonical spine, and jumps you there. Useful when you remember a spine label, not a page of text.

Proof of find

A gallery whose hash starts with many zero bits is exponentially rare — finding one is real proof-of-work, but anyone can verify it instantly by recomputing the hash. Click rarity or actions… → prospect to scan random galleries. When you walk into an uncommon one (8+ zero bits), it lands in trophies automatically. Share the link — whoever opens it can prove the find.
